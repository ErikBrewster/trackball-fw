# trackball-fw

This is a Circuit Python project for a Raspberry Pi Pico using a PMW3360 from Jack Industries
https://www.tindie.com/products/jkicklighter/pmw3360-motion-sensor/

This project is a very rough port of the existing Arduino PMW3360 from 
https://github.com/SunjunKim/PMW3360 and
https://github.com/mrjohnk/PMW3360DM-T2QU/tree/master/Arduino%20Examples/PMW3360DM-polling
